# CSS Color Codes Style Guide

This style guide contains a collection of the color codes used throughout the project.

## Colors

- **Primary Blue**: `#003366`
- **Primary Green**: `#007A3D`
- **Primary Red**: `#D32F2F`

---

## Background Colors

- **Light Background**: `#F5F5F5`
- **Dark Background**: `#111111`
- **Yellow**: `#FFD43A`
- **White2**: `#ebebeb`


---

## Text Colors

- **White Smoke**: `#f4f4f4`
- **Black**: `#212429`
- **Light Black**: `#7f7f7f`

---

## Border Colors

- **Light Border**: `#E0E0E0`
- **Dark Border**: `#BDBDBD`
